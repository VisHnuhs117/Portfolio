NearbyStoreApp(July 2025)

The Nearby Store App is an Android application built using Kotlin, Jetpack Compose, Firebase, and Google Maps API. It helps users locate nearby stores, view store details, and navigate to them with ease. Designed with modern Android development tools, the app provides a seamless and intuitive user experience.

Features:

  • Real-time location tracking using Google Maps.
  • Display nearby stores on a map.
  • Search for stores by name.
  • Cloud-based store data with Firebase Firestore.
  • Modern UI with Jetpack Compose.

TheNotesApp(June 2025)

The Notes App is a simple and efficient Android application developed using Kotlin that allows users to create, edit, delete, and organize notes. Designed with a clean user interface and seamless user experience, the app focuses on offline functionality using Room Database for local data storage.

Features:

  • Create & Edit Notes - Write and update text-based notes anytime.
  • Delete Notes - Remove unwanted notes with confirmation prompts.
  • Organized List View – View all notes in a structured format.
  • Search Functionality – Easily find notes with a simple keyword.
  • Offline Access – Fully functional without internet using Room DB.
